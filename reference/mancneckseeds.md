# Seed plane for the Neck Connective

These were generated by Chris Ordish of FlyEM and [passed to Greg
Jefferis on 18th April 2021 by Stuart
Berg](https://flyem-cns.slack.com/archives/C01BT2XFEEN/p1618779811015700).

## Usage

``` r
mancneckseeds
```

## Format

An object of class `data.frame` with 3781 rows and 3 columns.

## Details

Note that these positions are raw (8nm voxel) locations.

## See also

[`mancsomapos`](https://natverse.org/malevnc/reference/mancsomapos.md)
