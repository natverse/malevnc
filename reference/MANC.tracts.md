# Meshes of 22 MALEVNC tracts.

`MANC.tracts` contains mesh of 22 MANC tracts in non-symmetric space:
CVL_L, CVL_R, DLT_L, DLT_R, DLV_L, DLV_R, DMT_L, DMT_R, ITD_L, ITD_R,
MDA_L, MDA_R, type-I_MTD_L, type-I_MTD_R, type-II_MTD_L, type-II_MTD_R,
type-III_MTD_L, type-III_MTD_R, VLT_L, VLT_R, VTV_L, VTV_R

## Usage

``` r
MANC.tracts
```

## Format

An object of class `hxsurf` (inherits from `list`) of length 4.

## Examples

``` r
if (FALSE) { # \dontrun{
# Depends on nat
library(nat)
plot3d(MANC.tracts)
} # }
```
